Installation
============

Installation of :math:`\texttt{SEDA}` via GitHub:

>> git clone https://github.com/suarezgenaro/seda.git

.. code-block:: bash

    git clone https://github.com/suarezgenaro/seda.git
    cd seda
    python setup.py install
